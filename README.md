 # Intro To Excel 
 This is a mini project showcasing various skills learnt using Excel.

 The skills demostrated are as follows

 - Sum : This was used to calculate the total hours worked, taxable income and taxes to be paid.
- Average: This was used to get average of hours worked, taxable income and taxes to be paid.
- Max: This was used to get the highest hours worked, taxable income and taxes to be paid.
- Min: To get the lowest hours worked, taxable income and taxes to be paid.
- Count: Used to count the numbers of employees.
